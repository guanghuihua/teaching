## 适合用途

这个模板适合：

- 试卷讲评课
- 高一数学素养提升课
- 专题复习课
- 一题一页的例题讲解课件

## 风格特点

```text
ctexbeamer
16:9 宽屏
默认主题
衬线字体
无导航按钮
仅保留页码
题目先出现，答案和解析 pause 后出现
```

整体效果接近“投影版学案”，干净、克制，适合数学推导和试题讲评。

## 常用命令

```latex
\answer{答案内容}
\analysis{解析内容}
\proofblock{证明内容}
\methodnote{方法小结}
```

## 使用建议

每一道题尽量保持下面的结构：

```text
题目
必要图形或选项
\pause
答案
详解
方法点拨
```

讲评课不要只按题号推进。可以在若干题之后插入“方法小结页”，把题目上升为结构：

```text
这类题识别什么？
为什么用这个方法？
换一个条件还能不能用？
学生最容易错在哪里？
```

这和你的“套路与工具”教学思考是一致的：模板服务于题目讲解，但每次讲解都尽量把套路提升为工具。
