# 费曼式理解：不能创造就没有真正理解

## 原句

Richard Feynman:

```text
What I cannot create, I do not understand.
```

可以理解为：

```text
如果我不能把一个东西从头构建出来，我就没有真正理解它。
```

## Create 的含义

这里的 create 不是简单动手做，而是从基本起点出发，重新建造一个理论、证明或推导。

如果只能看懂别人写好的证明，却不能合上书自己重建，那么理解还不稳。

## 数学中的例子

读一个定理证明时，可能每一步都能跟上，合上书却写不出来。

这说明：

```text
我能验证这条路是对的，
但我还不知道为什么要走这条路。
```

真正理解意味着：

```text
知道每一步的动机
知道为什么这样而不是那样
知道如果走错会卡在哪里
能用自己的语言重新组织
能在白纸上重建
```

## 费曼式学习方法

遇到一个新理论或新结论时，可以尝试：

```text
先看结论
合上书自己推
推不出来再看别人怎么做
看完后重新自己走一遍
直到能独立完成
```

这比反复阅读更有效，因为它迫使大脑主动参与。

## 对自己的学习原则

学完一个东西后，不只问：

```text
我看懂了吗？
```

还要问：

```text
我能不能从头写出来？
我能不能讲给别人听？
我能不能解释每一步为什么出现？
我能不能换一个例子重新构造？
```

能建出来的部分，才是我真正掌握的。

建不出来的地方，就是下一步要学习的地方。

## 与自研式学习的关系

这和 [[主动钻研与自研式学习]] 是同一方向。

真正的理解不是搬运符号，而是参与知识生成。

数学学习中，最重要的不是“看过”，而是“能重建”。
