# 费力与痛苦：effort 和 suffering

## 核心区分

学数学对所有人都是费力的。

但“费力”和“痛苦”不是一回事。

用英语区分会更清楚：

```text
费力：effort / effortful
痛苦：suffering
```

## Effort

Effort 是一种客观消耗。

它描述的是大脑或身体正在为一个有难度的任务消耗能量。它本身是中性的，不带自我攻击。

例如：

```text
读证明读得慢
推导需要反复写
做题时需要集中注意力
半小时只能理解一个小步骤
```

这些都是 effort。

如果停在这里，这是正常学习。

## Suffering

Suffering 往往不是任务本身造成的，而是来自附加在任务上的心理故事。

例如：

```text
我怎么这么笨？
别人肯定很快就懂了。
我是不是没天赋？
我花了这么久还不会，说明我不适合。
```

同样是看不懂一个证明，如果只是困惑和疲劳，那是 effort；如果开始进行自我否定，那就变成 suffering。

## Pain is inevitable, suffering is optional

这个表达很精确：

```text
Pain 是不可避免的。
Suffering 是在 pain 之上添加的故事和评判。
```

数学学习中的困惑、疲劳、卡住，是 pain 或 effort。

真正让人逃避、拖延、刷手机、自我厌恶的，常常是 suffering。

## 一个具体场景

你坐下来学实分析，读了半小时发现一个定理证明看不懂。

到这里为止：

```text
大脑累
困惑
需要停下来
```

这是 effort。

但如果接下来想：

```text
高斯可能十分钟就懂了。
我果然没天赋。
我这种人不适合数学。
```

这就是 suffering。

## 自我提问

下次学数学感到吃力时，可以问：

```text
我现在感受到的是 effort，还是 suffering？
```

如果是 effort，说明我正在学习。

如果是 suffering，就观察一下脑子里正在讲什么故事，然后把注意力拉回题目本身。

## 对教学的启发

学生在数学中受伤，很多时候不是因为题目本身难，而是因为困难被解释成自我价值的失败。

教师要尽量减少这种 suffering：

```text
允许学生不会
允许学生试错
把错误当信息
把费力解释为学习的一部分
不要用羞辱感推动学习
```
