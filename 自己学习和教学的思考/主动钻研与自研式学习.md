## 一、原文提取

主动钻研。

在清华基科和巴黎综合理工念书时，发现几乎所有学习很好的人与其说是在“学习”，不如说是“自研”。课本上的大部分知识都是靠自己推出来的，除了最基本的数学定义、物理定义、实验规律外，各种次一级的公式、结论、理论、规律几乎都是靠自己推出来的。

有些非常精妙和复杂的过程可能自己想不到，但是在学习完之后也一定能自己从头推到尾，最后甚至能自己从头到尾把课本重新写出来。

人类获取知识的两个最伟大的能力是抽象能力和逻辑能力。

抽象能力是从看上去各不相同、千变万化的表象中提取出共性的抽象概念；逻辑是一套几乎百试百灵地分析、推导、归纳事物之间关系的方法论。

抽象能力和逻辑能力是人类总结知识、创造文明的核心能力，也是后人学习前人知识的核心能力。

学习的关键在于理解，而不是毫不理解地死记硬背一堆公式文本和题型套路。因为我们学习的知识都是对客观世界非常精炼、通用、有预见性的描述。简短的一个公式能描述千变万化的具体对象，只有利用抽象思维真正理解了这一个公式是怎么描述看上去各不相同的表象，利用逻辑能力从一个结论得出数不清的可靠结论，才能通用和万能地解决一切同类问题，而这种能力是没法靠死记硬背来得到的。

记忆力虽然重要但非常低等，跟没有智慧和文明的动物没有区别。普通人的记忆力完全足以掌握知识，真正导致学习能力差异的就是使用抽象能力和逻辑能力的人。

有的人虽然记忆能力非常强，但是如果因此就习惯于用记忆力偷懒，没能训练掌握抽象能力和逻辑能力，那么就算能死记硬背成百上千个结论，但还是没法跟相当于掌握了无穷多结论的、用抽象能力和思维能力彻底通关的学习者相比。

提升抽象能力和逻辑能力的最好方法就是使用和训练。最好的训练方法是解决具体问题，做题练习就是一种很好的解决问题的形式。实际上自己分析推导出新的知识也是一种很好的训练方式，哪怕这些知识实际上前人已经推出来了，但是自己独立地重新推出来，跟从书本上简单轻松地直接看过来是截然不同的。

自己独立地总结推理出来，经历了抽象、提炼、分析、推理等过程，对知识的方方面面都需要认真考虑，实际上是跟透彻、全面、严格地剖析和理解了概念。而不像简单看书那样容易浮于表面，只知道了文本信息，却无从得知隐藏在简略的文本背后的大量细节。

很多学习很好的人会说自己从不上辅导班，或者没有刷数不清的题目（但仍然不少），就是因为通过主动研究得来的知识远比从别人那里听来的知识要更详细和牢靠，并且学习研究的思考已经抵得上大量刷题的效果。这些都是真正从本质层面学会知识，并训练出利用知识解决问题的能力，而不是肤浅表面地只会做题和死记硬背题型套路。

真正学习出色的学生，越是学习优秀，越是早地形成了自学与自研的习惯。

自研对研究生阶段是必须的，因为研究生的任务就是学习细分领域的知识并作出独创性成果。

自研对大学生也非常重要，因为大学知识的原理性、逻辑性、系统性已经非常强，不能自己从头到尾全部推一遍的话很难掌握，只是因为现在很多大学和课程对学生要求太低，所以才能没学懂背背公式水过。

为什么很多名牌大学的本科生就能在科研中作出突出的成果？因为本来大学知识的学习就跟科研是无缝衔接的，学习过程中就习惯于自己分析和推理出结论，很容易就能顺滑到细分领域的研究和解决问题上去。

自研几乎是高中学神学霸的专属，因为高中知识已经初步有了原理、逻辑和体系的雏形，只是因为数学等基础知识还不够，所以显得不够专业。但是如果能用自研的方法学习，几乎就能吊打被动汲取和死命刷题的同学，不仅成绩突出还看上去很轻松，实际上是把努力花在了刀刃上。

小学初中阶段极难见到自研的学生，因为小孩的天性使然，几乎只有极少的小孩愿意研究知识。再加上过度焦虑和灌输知识的家长老师，就算有这样的苗头也很容易被被动灌输大量知识给掐灭。

自发研究的主动性很难有固定的培养路径，极小的一部分是天生的，大多数是在小时候的生活、玩耍、寂寞中被“逼迫”出来的。

另外，主动研究的思维特点也不只限于语数外理化生史地政之类的课本。很多人对课本知识没什么兴趣，但是对生活和社会中的其他方面有非常大的兴趣，虽然学历不够光鲜，但是也在自己喜爱的领域做出了厉害的成就。

## 二、核心观点整理

这篇帖子的核心不是简单说“自学很重要”，而是把学习区分成两种状态：

```text
被动接收：听别人讲，记住结论，套用题型。
主动自研：自己抽象、推导、验证、重建知识。
```

作者认为，真正高质量的学习不是把课本内容“看懂”，而是能够把知识重新推出来。也就是说，学习者不是在复制结论，而是在复现知识生成的过程。

其中最关键的两种能力是：

```text
抽象能力：从不同表象中提取共同结构。
逻辑能力：从已有条件和结论出发，推出新的可靠结论。
```

这与数学学习高度相关。数学公式、定理和方法本来就是高度压缩的结果。学生如果只是记住它们，就只能在熟悉题型中使用；如果能理解它们如何生成、为什么成立、能解决什么结构，就会获得更强的迁移能力。

## 三、对高中数学教学的启发

这篇帖子对高中数学教学有很强的提醒：教师不能只把学生训练成“会听课、会记笔记、会套题”的人。

高中阶段虽然还没有进入真正的专业研究，但已经具备了让学生进行“小型自研”的条件。比如：

```text
从特殊例子猜一般结论
从图形观察抽象代数关系
从一道母题改编出变式
从公式出发重新推导相关公式
从错误解法中发现条件边界
从多个解法中比较结构差异
```

这些活动不是额外的花活，而是训练抽象能力和逻辑能力的具体方式。

## 四、与刷题的关系

这篇帖子并不等于否定做题。相反，它承认做题是训练抽象能力和逻辑能力的一种很好形式。

关键在于：学生是在“刷题”，还是在“借题自研”。

```text
刷题：做完一道题，记住一种做法。
自研：做完一道题，追问它为什么这样做，还能怎样变。
```

如果做题只是为了积累题型，学生容易越来越依赖记忆。如果做题之后能整理结构、复盘入口、寻找变式、解释方法边界，那么做题就会转化为研究。

这和之前的教学思考 [[套路与工具的边界在哪里？]] 可以接上：

```text
套路是记忆层面的压缩。
工具是理解层面的生成。
自研则是让学生亲自参与这种生成。
```

## 五、课堂中可以怎样培养“自研感”

高中课堂不可能完全变成学生自由研究，但可以在很多细节中增加“自研感”。

例如：

```text
讲公式前，先让学生猜它可能长什么样。
给定义后，先让学生判断几个正例和反例。
讲例题前，先让学生说可能的入口。
讲完解法后，让学生解释每一步的必要性。
做完母题后，让学生改一个条件并判断结论是否还成立。
讲评错题时，让学生自己定位错误来自概念、运算还是方法选择。
```

这些设计的意义在于，让学生不只是跟着教师走，而是不断产生自己的判断。

## 六、阶段性结论

这篇帖子给我的核心提醒是：

```text
好的数学学习，不只是获得结论，
而是经历结论被生成出来的过程。
```

对教师来说，真正有价值的课堂不只是“我讲得清楚”，而是：

```text
学生有没有机会猜想？
有没有机会推导？
有没有机会犯错？
有没有机会修正？
有没有机会把知识重新组织成自己的东西？
```

如果学生能逐渐形成这种主动钻研的习惯，那么他做题时就不只是积累套路，而是在训练抽象、逻辑和迁移能力。这种能力比单纯多刷几套题更长久，也更接近数学学习本身。
